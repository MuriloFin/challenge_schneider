Sobre o Projeto

Idealizador e Criador do Projeto

O presente projeto foi desenvolvido por Murilo Favoretto Finardi, aluno do curso de Engenharia de Software da FIAP.

Recursos Utilizados

Para o desenvolvimento do projeto Green Path foram utilizadas as presentes ferramentas:

HTML

Além de servir como estrutura para todo o projeto, o HTML foi utilizado para criar o principal mecanismo deste site, o formulário chamado de Checklist, no qual o colaborador responderá de maneira rápida e efetiva.

CSS

O CSS foi o grande responsável pela estilização do site, de maneira que estabeleceu margens e fontes para os textos, além de cores e bordas

Javascript

O Javascript foi utilizado para dar real funcionalidade ao formulário, de maneira que através de uma Function tornou-se possível calcular se o colaborador está agindo de maneira sustentável, ou se será necessário repensar algumas práticas diárias.

Bootstrap

O Bootstrap foi o framework escolhido para estilizar alguns botões do site, para que o usuário tivesse uma melhor experiência.
